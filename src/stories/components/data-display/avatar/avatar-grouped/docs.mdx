<Title>Grouped avatars</Title>
<Subtitle><code>AvatarGroup</code> renders its children as a stack.</Subtitle>

### Import

```jsx
import Avatar from 'qasir-ui/data-display/Avatar';
import AvatarGroup from 'qasir-ui/data-display/AvatarGroup';
```

<h3 style={{marginBottom: 20}}>Usage</h3>

<AvatarGroup />

```jsx
<AvatarGroup max={4}>
 <Avatar alt="Remy Sharp" src="/static/images/avatar/1.jpg" />
 <Avatar alt="Travis Howard" src="/static/images/avatar/2.jpg" />
 <Avatar alt="Cindy Baker" src="/static/images/avatar/3.jpg" />
 <Avatar alt="Agnes Walker" src="/static/images/avatar/4.jpg" />
 <Avatar alt="Trevor Henderson" src="/static/images/avatar/5.jpg" />
</AvatarGroup>
```
