# Chip

<Subtitle>
Chips are compact elements that represent an input, attribute, or action.
</Subtitle>

### Import
```js
import Chip from 'qasir-ui/data-display/Chip';
```

### Props
```html
(Required if you want to change Chip Background and Font Color)
```
```js
customBackground = String
customFont = String 
```

### Usage
```jsx
<Chip customBackground="#FFFF00" customFont="#008000" label="Custom Color"/>
```
