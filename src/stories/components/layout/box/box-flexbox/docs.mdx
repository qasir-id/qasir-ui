<Title>Flexbox</Title>
<Subtitle>Quickly manage the layout, alignment, and sizing of grid columns, navigation, components, and more with a full suite of responsive flexbox utilities.</Subtitle>

### Import

```jsx
import Box from 'qasir-ui/layout/Box';
```

### Usage 

### Properties for the Parent

<BoxFlex />

