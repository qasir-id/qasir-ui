<p style={{fontSize: 18, fontFamily: ['"Montserrat"', 'sans-serif'].join(',')}}>The Box component serves as a wrapper component for most of the CSS utility needs.</p>

### Import

```js

```

### Usage

```html
<Box component="span" m={1}>
  <Button />
</Box>
```
