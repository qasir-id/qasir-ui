<Title>Palette</Title>
<Subtitle>Convey meaning through color with a handful of color utility classes. Includes support for styling links with hover states, too.</Subtitle>

### Import

```jsx
import Box from 'qasir-ui/layout/Box';
```

### Usage

<BoxPaletteColor />

```jsx
<Box color="#4F94FC">blue50</Box>
```

<BoxPaletteBackgroundColor />

```jsx
<Box bgcolor="#4F94FC" color="#ffffff" p={2}>blue50</Box>
```
