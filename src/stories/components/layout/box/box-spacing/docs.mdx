### Notation
<Subtitle>The space utility converts shorthand margin and padding props to margin and padding CSS declarations. The props are named using the format <code>property</code><code>sides</code>.</Subtitle>
<p>Where property is one of:</p>
<ul>
<li><code>m</code> - for classes that set <b>margin</b></li>
<li><code>p</code> - for classes that set <b>padding</b></li>
</ul>
<p>Where sides is one of:</p>
<ul>
<li><code>t</code> - for classes that set <b>margin-top</b> or <b>padding-top</b></li>
<li><code>b</code> - for classes that set <b>margin-bottom</b> or <b>padding-bottom</b></li>
<li><code>l</code> - for classes that set <b>margin-left</b> or <b>padding-left</b></li>
<li><code>r</code> - for classes that set <b>margin-right</b> or <b>padding-right</b></li>
<li><code>x</code> - for classes that set both <b>*-left</b> and <b>*-right</b></li>
<li><code>y</code> - for classes that set both <b>*-top</b> and <b>*-bottom</b></li>
<li>blank - for classes that set a margin or padding on all 4 sides of the element</li>
</ul>
