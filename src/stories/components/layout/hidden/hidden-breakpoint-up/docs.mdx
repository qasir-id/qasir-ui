<Title>Breakpoint Up</Title>
<Subtitle>Using any breakpoint <code>up</code> property, the given <em>children</em> will be hidden <em>at or above</em> the breakpoint.</Subtitle>

### Import

```jsx
import Hidden from 'qasir-ui/layout/Hidden';
```

### Usage

<HiddenBreakpointUp />

  code={dedent`

    </Hidden>
    <Hidden lgUp>
      <span>lgUp</span>
    </Hidden>
    <Hidden xlUp>
      <span>xlUp</span>
    </Hidden>
  `}
