<Title>Hidden</Title>
<Subtitle>Quickly and responsively toggle the visibility value of components and more with the hidden utilities.</Subtitle>

### How it works
<Subtitle>
  Hidden works with a range of breakpoints e.g. <code>xsUp</code> or <code>mdDown</code>, or one or more breakpoints e.g. <code>only='sm'</code> or <code>only={['md', 'xl']}</code>.
Ranges and individual breakpoints can be used simultaneously to achieve very customized behavior.
The ranges are inclusive of the specified breakpoints.
</Subtitle>

```sh 
innerWidth  |xs      sm       md       lg       xl
            |--------|--------|--------|--------|-------->
width       |   xs   |   sm   |   md   |   lg   |   xl

smUp        |   show | hide
mdDown      |                     hide | show
```

### Import

```jsx
import Hidden from 'qasir-ui/layout/Hidden';
```

### Usage

```jsx
<Hidden xsUp>...</Hidden>
```
