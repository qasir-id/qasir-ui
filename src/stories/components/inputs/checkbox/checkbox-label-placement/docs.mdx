<Title>Label placement</Title>
<Subtitle>
You can change the placement of the label:
</Subtitle>

### Import

```js
import FormControlLabel from 'qasir-ui/inputs/FormControlLabel';
import Checkbox from 'qasir-ui/inputs/Checkbox';
```

<h3 style={{marginBottom: 20}}>Usage</h3>

<CheckboxLabelStart />


```jsx
<FormControlLabel
  value="start"
  control={<Checkbox color="primary" />}
  label="Start"
  labelPlacement="start"
/>
```
