<Title>Sizes Buttons</Title>
<Subtitle>Fancy larger or smaller buttons? Use the size property.</Subtitle>

### Import

```js
import Button from 'qasir-ui/inputs/Button';
```

### Usage

<ButtonLarge style={{marginTop: 20}} />

```html
<Button variant="contained" color="primary" size="large">Large</Button>
```

<ButtonMedium />

```html
<Button variant="contained" color="primary">Medium</Button>
```

<ButtonSmall />

```html
<Button variant="contained" color="primary" size="small">Small</Button>
```
