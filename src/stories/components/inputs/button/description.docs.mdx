# Button
<Subtitle>Buttons are action items that can initiate a certain action or change state, they can have customisable background, icons and text. Buttons should be easy to discover and clearly indicate the intended action.</Subtitle>

### Import

```js
import Button from 'qasir-ui/inputs/Button';
```

### Usage

```html
<Button variant="contained" color="primary">
  Hello World
</Button>
```
