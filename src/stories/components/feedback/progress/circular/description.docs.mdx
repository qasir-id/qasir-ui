<Subtitle>Progress indicators commonly known as spinners, express an unspecified wait time or display the length of a process. The animation works with CSS, not JavaScript.</Subtitle>

<p>Progress indicators inform users about the status of ongoing processes, such as loading an app, submitting a form, or saving updates. They communicate an appâ€™s state and indicate available actions, such as whether users can navigate away from the current screen.</p>

<ul>
<li><strong>Determinate</strong> indicators display how long an operation will take.</li>
<li><strong>Indeterminate</strong> indicators visualize an unspecified wait time.</li>
</ul>

<p>When displaying progress for a sequence of processes, indicate overall progress rather than the progress of each activity.</p>
