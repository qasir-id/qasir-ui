<Title>Recommended libraries</Title>
<Subtitle>A list of libraries that we recommend to use.</Subtitle>

### CSS

<ul>
  <li>
    <p><a class="link" target="_blank" href="https://sass-lang.com/">SCSS/SASS</a> is a CSS pre-processor with syntax advancements and variable, functions, etc. Learn more about how to use scss with create-react-app.</p>    
  </li>
  <li>
    <p><a class="link" target="_blank" href="https://styled-components.com/">Styled Components</a> is the result of wondering how we could enhance CSS for styling React component systems.</p>
  </li>
  <li>
    <p><a class="link" target="_blank" href="https://cssinjs.org">JSS</a> is an authoring tool for CSS which allows you to use JavaScript to describe styles in a declarative, conflict-free and reusable way. It can compile in the browser, server-side or at build time in Node.</p>
  </li>
</ul>

### Application state handling

<ul>
  <li>
    <p><a class="link" target="_blank" href="https://redux.js.org/">Redux</a> is a predictable state container for JavaScript apps. It helps you write applications that behave consistently, run in different environments (client, server, and native), and are easy to test.</p>
  </li>
  <li>
    <p><a class="link" target="_blank" href="https://react-redux.js.org/">React Redux</a> is maintained by the Redux team, and kept up-to-date with the latest APIs from Redux and React.</p>
  </li>
  <li>
    <p><a class="link" target="_blank" href="https://github.com/reduxjs/redux-thunk">Redux Thunk</a> is middleware that allows you to return functions, rather than just actions, within Redux.</p>
  </li>
  <li>
    <p><a class="link" target="_blank" href="https://github.com/rt2zz/redux-persist">Redux Persist</a> is a library allowing to save the redux store in the local storage of your browser.</p>
  </li>
</ul>

### Forms

<ul>
  <li>
    <p><a class="link" target="_blank" href="https://formik.org/">Formik</a> is a small group of React components and hooks for building forms in React and React Native.</p>
  </li>
  <li>
    <p><a class="link" target="_blank" href="https://github.com/jquense/yup">Yup</a> is a dead simple Object schema validation</p>
  </li>  
</ul>

### Tables

<p><a class="link" target="_blank" href="https://material-ui.com/components/data-grid/">Material UI Table</a> is a fast and extendable data table and data grid for React.</p>

### Charts

<p><a class="link" target="_blank" href="https://recharts.org/en-US/">Recharts</a> is a set of modular charting components for React.</p>

### Maps

<ul>
  <li>
    <p><a class="link" target="_blank" href="https://www.react-simple-maps.io/">react-simple-maps</a> is a map component to generate simple political maps.</p>
  </li>
  <li>
    <p><a class="link" target="_blank" href="https://github.com/visgl/react-map-gl">react-map-gl</a> for maps using mapbox.gl</p>
  </li>  
  <li>
    <p><a class="link" target="_blank" href="https://developers.arcgis.com/javascript/">ArcGIS API for JavaScript</a> for complex maps with custom layers</p>
  </li>  
</ul>

### Fuzzy Searching

<p><a class="link" target="_blank" href="https://fusejs.io/">Fuse.js</a> is a powerful, lightweight fuzzy-search library, with zero dependencies.</p>

### Polyfills

<p>Polyfills are needed for IE browsers. We recommend <a class="link" target="_blank" href="https://babeljs.io/docs/en/babel-preset-env">babel-preset-env</a> for it.</p>

### Linting & naming

<p>Airbnb provides a <a class="link" target="_blank" href="https://github.com/airbnb/javascript">useful styling guide</a> on how to style JavaScript and react code.</p>

### React Framework

<ul>
  <li>
    <p><a class="link" target="_blank" href="https://nextjs.org/">Next.js</a> is a modern front-end development web framework that enables functionality such as server-side rendering and generating static websites for React based web applications. The combination of server side rendering and client side hydration allows the creation of very performant websites. It can be easily connected to headless content management systems like Drupal or Wordpress.</p>
  </li>
  <li>
    <p><a class="link" target="_blank" href="https://material-ui.com/">Material-UI</a> is an open-source project that features React components that implement Google's Material Design.</p>
  </li>  
</ul>
