### Test with Google Chrome

<p>The Network tab in <code>Chrome DevTools</code> has an option to faux throttle your network, so you can experience what your users might experience visiting your application on 3G, 2G and GPRS connections. Besides the obvious, itâ€™s also useful for visualising how the page load.</p>

<ul>
  <li>
    <p>
      Open DevTools (<code>cmd</code> + <code>alt</code> + <code>i</code>)
    </p>
  </li>
  <li>
    <p>
      Click the <code>Network</code> tab
    </p>
  </li>
  <li>
    <p>Select which type of connection you want to imitate</p>
  </li>
  <li>
    <p>
      Reload the page to see assets downloading at that connection speed
    </p>
  </li>
</ul>

<p>You can make a custom Network Throttling Profile vor VSAT connections by clicking the option at the very top of the dropdown menu and use the values above.</p>
